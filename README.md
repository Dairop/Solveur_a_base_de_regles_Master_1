Projet universitaire

Solveur à base de règles.
Règles fournies (en tant qu'exemple) permettent de détecter de potentielles maladies en fonction de symptômes.
Le programme peut appliquer n'importe quelle type de règle, voir les règles fournies dans /Moteur1.txt

Fonctionnalités:
- Variables de type "enum" ou binaires -> ensemble de recherche restreint
- Vérification de la cohérence (à aucun moment il n'est possible de déduire !A et A)
- Chaînage avant
- Chaînage arrière
- Chaînage par paquets
- Interface utilisateur interactive (Swing)
